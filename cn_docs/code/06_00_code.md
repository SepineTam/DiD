# Stata 代码

本部分旨在涵盖来自各种包的Stata估计命令。

所有代码使用以下符号集：

| 符号 | 描述 | 
| $$ i $$ | 面板标识符 |
| $$ t $$ | 时间变量 |
| $$ Y $$ | 结果变量 |
| $$ D $$ | 如果为处理观测值则=1 |
| $$ \epsilon $$ | 误差项 |
