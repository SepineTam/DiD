- 更多信息：不在SSC上。直接从GitHub安装。需要额外的`gwtmean`包。



## 安装和选项

```stata
net install fect, from("https://raw.githubusercontent.com/xuyiqing/fect_stata/master/") replace
ssc install _gwtmean, replace  // 依赖包
```

查看帮助文件：

```stata
help fect
```
